## 核心思维转变：变量不是箱子

**C#：变量是一个固定大小的箱子。**

```csharp
int age = 25;        // 箱子标签写死了：只能装 int
age = "hello";       // 编译报错——往 int 箱子里塞字符串？
```

**Python：变量是一个便签，贴在值上。**

```python
age = 25             # 给整数 25 贴了个标签叫 age
age = "hello"        # 把 age 标签撕下来，贴到字符串 "hello" 上
```

| | C# | Python |
|--|-----|--------|
| 变量是什么？ | 固定大小的容器 | 贴在对象上的名字 |
| 赋值是什么？ | 把值放进容器 | 把标签贴到对象上 |
| 类型由谁决定？ | 变量声明时固定 | 运行时由实际对象决定 |

**推论一：`b = a` 不复制数据**

```python
a = [1, 2, 3]
b = a            # b 和 a 是同一对象的两个标签
a.append(4)
print(b)         # [1, 2, 3, 4] —— 同一个对象，当然能看到
```

C# 的引用类型和这个行为完全一样。但 C# 还有**值类型**（struct、int、bool 等），Python 没有——在 Python 里，**所有东西都是对象，所有赋值都是贴标签**。

**推论二：变量没有类型，对象才有**

```python
x = 5          # x 指向一个 int 对象
x = "hello"    # 现在 x 指向一个 str 对象
```

"x 的类型"这种说法在 Python 里不准确。准确的说法是："x 当前引用的对象的类型是 int/str"。

## 强类型 vs 动态类型

很多初学者把这两个混为一谈。它们是两个独立维度：

**维度一：强类型 vs 弱类型**

```
Python:  "hello" + 5   → 报错（不会偷偷转）
JavaScript: "hello" + 5 → "hello5"（偷偷转成字符串）
```

Python 是**强类型**——不同类型做不兼容的操作会直接报错，不会悄悄转换。

**维度二：静态类型 vs 动态类型**

```
C#:    int x = 5; x = "hello"  → 编译报错（编译时就检查）
Python: x = 5; x = "hello"     → 完全合法（运行时才知道）
```

Python 是**动态类型**——变量可以随时绑定到不同类型的对象上。

**所以 Python 的准确定位是：强类型 + 动态类型。**

```python
# 强类型：不允许跨类型运算
"hello" + 5       # TypeError
5 + "hello"       # TypeError

# 动态类型：变量可以换类型
x = 10
x = [1, 2, 3]     # 完全合法
x = lambda y: y   # 甚至变成函数
```

## 基础数据类型

### 不可变类型

```python
age = 25           # int
price = 3.14       # float
name = "Bob"       # str
is_ok = True       # bool
nothing = None     # NoneType
```

### 一个容易忽略的点：bool 其实是 int

```python
print(True + True)     # 2
print(True * 5)        # 5
print(False * 5)       # 0
print(True == 1)       # True
```

这是历史遗留——bool 继承自 int。C# 不会让你 `true + true`，但 Python 可以。

**工程建议**：永远不要写 `if x == True`，用 `if x` 就行。

### None 的处理：为什么必须用 is

```python
result = None

# 检查 None
if result is None:      # ✓ 推荐
    ...

if result == None:      # × 不推荐，虽然能用
    ...
```

**原因：**

```python
class CustomNull:
    def __eq__(self, other):
        return True     # 我说和 None 相等，你怎么办？

x = CustomNull()
print(x == None)    # True —— 但实际上 x 不是 None
print(x is None)    # False —— is 不会骗你
```

`==` 调用的是值的 `__eq__` 方法，可以被重写。
`is` 比较的是内存地址，不可能被重写。

整个 Python 里只有一个 None 对象，所以 `is None` 既准确又高效。

## 字符串

### f-string（最常用的字符串格式化）

```python
name = "Alice"
age = 25

msg = f"My name is {name} and I'm {age}"
# "My name is Alice and I'm 25"
```

为什么 f-string 比 `+` 拼接好？

```python
# 不要这样
msg = "My name is " + name + " and I'm " + str(age)

# 应该这样
msg = f"My name is {name} and I'm {age}"
```

| | `+` 拼接 | f-string |
|--|---------|----------|
| 可读性 | 差（断句、拼写） | 好（直接嵌入） |
| 转换 | 必须手动 `str()` | 自动调 `__format__` |
| 表达式 | 不支持 | 支持任意表达式 |

```python
# f-string 支持表达式
f"2 + 2 = {2 + 2}"                # "2 + 2 = 4"
f"{'hello'.upper()}"              # "HELLO"
f"{3.14159:.2f}"                  # "3.14"（保留两位小数）
f"{'hello':>10}"                  # "     hello"（右对齐）
```

## 条件与循环

### if 语句

```python
age = 20

if age >= 18:
    print("成年人")

# elif，不是 else if
if age < 13:
    print("儿童")
elif age < 20:
    print("青少年")
else:
    print("成年人")

# 三元表达式
status = "成年" if age >= 18 else "未成年"
```

### 循环：Python 只有 foreach

```python
# Python 没有 C# 的 for(int i=0; i<n; i++)，用 range 替代
for i in range(5):            # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):         # 1, 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2):     # 0, 2, 4, 6, 8（步长2）
    print(i)

# foreach 模式
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(fruit)

# while 和 C# 一样
count = 0
while count < 5:
    print(count)
    count += 1

# break / continue 和 C# 一样
for i in range(10):
    if i == 3:
        continue    # 跳过 3
    if i == 7:
        break       # 遇到 7 停止
    print(i)
```

## 函数

```python
# def，不需要返回值类型
def greet(name):
    return f"Hello, {name}!"

# 类型注解（推荐，但不强制）
def add(a: int, b: int) -> int:
    return a + b

# 默认参数值
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

# 多返回值（Python 特色）
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder     # 返回元组

q, r = divide(17, 5)              # 拆包接收
print(q, r)                       # 3, 2
```

## 快速对照表

| C# | Python |
|-----|--------|
| `int x = 5;` | `x = 5` |
| `string s = "hi";` | `s = "hi"` |
| `bool flag = true;` | `flag = True` |
| `var list = new List<int>();` | `list = []` |
| `for (int i=0; i<10; i++)` | `for i in range(10):` |
| `foreach (var item in list)` | `for item in list:` |
| `if (x > 0) { }` | `if x > 0:` |
| `return x + y;` | `return x + y` |
| `void DoIt() { }` | `def do_it():` |
