## 核心差异：C# vs Python 思维模式

**C# 是静态强类型语言：**
```csharp
string name = "Bob";
int age = 25;
List<string> names = new List<string>();
```

**Python 是动态类型语言：**
```python
name = "Bob"          # 不用声明类型
age = 25              # 同一变量可以随时换成其他类型
names = []            # 空列表，不指定类型
```

**核心区别：Python 靠缩进（indentation）而不是大括号来定义代码块。**

## 变量与类型

### Python 的类型系统

```python
# 整数
age = 25

# 浮点数
price = 19.99

# 字符串
greeting = "Hello, World!"
name = 'Bob'          # 单引号双引号都行

# 布尔值
is_active = True      # 注意大写 Ture/False，不是 true/false

# 多行字符串
description = """
    这是多行字符串，
    类似 C# 的 verbatim string (@"...")。
"""

# None（空值）= C# 的 null
result = None
```

### 类型注解（可选，但推荐）

```python
# Python 3.5+ 支持类型注解，类比 C# 的强类型声明
name: str = "Bob"
age: int = 25
prices: list[float] = [19.99, 29.99, 39.99]

def greet(name: str) -> str:
    return f"Hello, {name}!"
```

**为什么有用：** 类型注解让代码更清晰，配合 IDE（VS Code + Pylance）可以提供和 C# 类似的智能提示。

## 字符串

### 基础操作

```python
name = "Alice"

# 字符串拼接
greeting = "Hello, " + name + "!"

# f-string（类似 C# 的 $"" 插值，最常用）
greeting = f"Hello, {name}!"

# 格式化（类似 string.Format）
greeting = "Hello, {}!".format(name)

# f-string 进阶
price = 19.99
msg = f"{name} bought item for ${price:.2f}"  # Alice bought item for $19.99
```

### 常用方法

```python
text = "  Hello, World!  "

text.strip()           # "Hello, World!" （去首尾空格）
text.lower()           # "  hello, world!  "
text.upper()           # "  HELLO, WORLD!  "
text.replace("World", "Python")  # "  Hello, Python!  "
text.split(",")        # ["  Hello", " World!  "]

# 包含判断
"Hello" in text        # True（类似 C# 的 string.Contains）
```

## 条件与循环

### if 语句

```python
age = 20

# 单条件
if age >= 18:
    print("成年人")

# if-else（注意：不需要括号，用 elif）
if age < 13:
    print("儿童")
elif age < 20:
    print("青少年")
else:
    print("成年人")

# 三元表达式（类似 C# 的 condition ? a : b）
status = "成年" if age >= 18 else "未成年"
```

### 循环

```python
# for 循环（类似 C# 的 foreach）
fruits = ["苹果", "香蕉", "橙子"]

for fruit in fruits:
    print(fruit)

# range（类似 C# 的 Enumerable.Range）
for i in range(5):       # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):    # 1, 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8（步长2）
    print(i)

# while 循环
count = 0
while count < 5:
    print(count)
    count += 1

# break / continue（和 C# 一样）
for i in range(10):
    if i == 3:
        continue    # 跳过 i=3
    if i == 7:
        break       # 遇到 7 停止
    print(i)
```

## 函数

### 基本定义

```python
# 定义函数（def = define）
def greet(name):
    return f"Hello, {name}!"

# 带类型注解
def add(a: int, b: int) -> int:
    return a + b

# 默认参数值（类似 C# 默认参数）
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

# 调用
result = greet("Bob")              # Hello, Bob!
result = greet("Bob", "Hi")       # Hi, Bob!
```

### 多返回值（Python 特有）

```python
# C# 需要 out 参数或元组，Python 直接返回多个值
def divide(a, b):
    quotient = a // b      # 整除
    remainder = a % b      # 取余
    return quotient, remainder

q, r = divide(17, 5)      # q=3, r=2
```

### 可变参数

```python
def sum_all(*numbers):     # *args，类似 params
    total = 0
    for n in numbers:
        total += n
    return total

print(sum_all(1, 2, 3))    # 6
print(sum_all(1, 2, 3, 4, 5))  # 15

# 关键字参数 **kwargs（类似命名参数集合）
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="Bob", age=25)
# name: Bob
# age: 25
```

## 快速对照表

| C# 语法 | Python 语法 |
|---------|-------------|
| `int x = 5;` | `x = 5` |
| `string s = "hi";` | `s = "hi"` |
| `bool flag = true;` | `flag = True` |
| `var list = new List<int>();` | `list = []` |
| `for (int i=0; i<10; i++)` | `for i in range(10):` |
| `foreach (var item in list)` | `for item in list:` |
| `if (x > 0) { }` | `if x > 0:` |
| `string.Format("{0}", x)` | `f"{x}"` |
| `return x + y;` | `return x + y` |
| `void DoIt() { }` | `def do_it():` |
