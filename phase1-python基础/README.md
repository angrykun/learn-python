
## 每节结构

每节包含：
1. **核心概念** — 从根因出发，类比 C# 解释为什么这样设计
2. **代码示例** — 对照 C# 和 Python 的等价写法
3. **工程要点** — 实际开发中的坑和建议
4. **本节练习** — 严格只涉及本节概念，不超纲
5. **快速理解检测** — 学完后自测几个问题
